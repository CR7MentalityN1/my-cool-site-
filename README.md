# StudyConnect

Платформа для поиска совместных учебных проектов среди студентов университета.

## Описание

StudyConnect - это современное веб-приложение, которое помогает студентам находить единомышленников для совместной работы над учебными проектами. Платформа включает:

- Систему регистрации и аутентификации
- Личные профили студентов с информацией о навыках и проектах
- Ленту студентов с возможностью фильтрации по факультету и курсу
- Чат в реальном времени для общения
- Адаптивный дизайн для всех устройств

## Технологический стек

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **База данных**: Supabase (PostgreSQL)
- **Аутентификация**: Supabase Auth
- **Real-time**: Supabase Realtime
- **Иконки**: Lucide React

## Установка и запуск

### Предварительные требования

- Node.js (версия 18 или выше)
- npm или yarn
- Аккаунт Supabase (бесплатный)

### Шаг 1: Клонирование репозитория

```bash
git clone <your-repo-url>
cd project
```

### Шаг 2: Установка зависимостей

```bash
npm install
```

### Шаг 3: Настройка Supabase

1. Создайте бесплатный аккаунт на [supabase.com](https://supabase.com)
2. Создайте новый проект
3. Перейдите в Settings → API
4. Скопируйте `Project URL` и `anon public` ключ

### Шаг 4: Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

Заполните файл `.env` вашими данными из Supabase:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### Шаг 5: Настройка базы данных

База данных уже настроена через миграции Supabase. Таблицы `profiles` и `messages` были созданы автоматически вместе с необходимыми политиками безопасности (RLS).

### Шаг 6: Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

## Функциональность

### Регистрация и вход

- Регистрация по email и паролю
- Вход в систему
- Автоматическое создание профиля при регистрации

### Профиль студента

- Имя и фамилия
- Аватар (через URL)
- Факультет (выбор из списка)
- Курс (1-4)
- Навыки (список через запятую)
- Описание желаемого проекта
- Контакты для связи

### Лента студентов

- Просмотр всех зарегистрированных студентов
- Фильтрация по факультету
- Фильтрация по курсу
- Поиск по имени, навыкам и описанию проекта
- Модальное окно с полной информацией о студенте

### Чат

- Общий чат для всех пользователей
- Сообщения в реальном времени
- Отображение имени и аватара отправителя
- Последние 30 сообщений

## Безопасность

Приложение использует Row Level Security (RLS) в Supabase:

- Пользователи могут просматривать все профили
- Пользователи могут редактировать только свой профиль
- Пользователи могут читать все сообщения
- Пользователи могут отправлять сообщения только от своего имени
